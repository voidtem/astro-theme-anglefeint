# Release Notes: 0.2.3

## Highlights

This is a hotfix for the 0.2.2 pagination migration. It restores installed-package resolution for the new pagination utility subpaths used by the starter branch.

## Major Themes

### Installed-package export compatibility

- added explicit package exports for:
  - `@anglefeint/astro-theme/utils/number`
  - `@anglefeint/astro-theme/utils/pagination`
  - `@anglefeint/astro-theme/utils/pagination-style`
- kept the pagination component itself package-owned while making the new util imports resolvable in real starter installs under `node_modules`

## Migration Notes

- no config migration is required
- starter sync is required so the starter branch can be regenerated against the fixed published package version

## Starter Sync

- starter sync is required for this release

## Validation Summary

- `npm run check:docs`
- `npm run check:adapters`
- `npm run check`
- starter installed-package validation via `npm run release:starter`
